Welcome to the BellMTS Network Approval Interval Tool

This tool is designed to assist the Provisioner in determining lead times for various designs and approvals, helping set an appropriate construction start date when submitting a request through Access Support.

The dates provided are based on historical norms but may vary due to internal and external factors. These timelines represent the minimum lead times and are intended for urgent requests. If you are able to extend these dates further, please do so whenever possible.
